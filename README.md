# HM-PGS

This repository is the implementation of HM-PGS

# Required packages

torch == 1.5.0+cpu

torch-geometric == 1.4.3 

numpy == 1.18.1

pandas == 1.0.1

sklearn == 0.22.1

# Usage

train and test 

```python train.py```

predict

```python predict.py --predict --model_path "saved/HM_PGS/final.model"```
